The Rules

  • Vo výslednom kóde nesmiete používať premenné (ak premenné už poznáte).
  • Pri preklade kódu sú použité nasledovné prepínače: -std=c11 -Wall -Werror -lsuperkarel -lcurses.
  • Pri implementácii môžete použiť knižnicu superkarel.h.
  • Pri práci môžete použiť Referenčnú príručku ku knižnici robota Karla.
  • K dispozícii máte aj celý obsah stránky predmetu Základy algoritmizácie a programovania.

Uloha 1

Funkcia void pick_all() (1b)

Vytvorte funkciu void pick_all(), pomocou ktorej robot Karel pozbiera všetky značky na danej pozícii.

Return

Funkcia nevráti žiadnu hodnotu.

Funkcia void put_all() (1b)

Vytvorte funkciu void put_all(), pomocou ktorej robot Karel vyloží všetky značky z batoha.

Return

Funkcia nevráti žiadnu hodnotu.

Funkcia void find_north_east() (2b)

Vytvorte funkciu void find_north_east(), ktorá premiestni robota Karla do pravého horného rohu sveta.

Return

Funkcia nevráti žiadnu hodnotu.

Počiatočná situácia

Na začiatku sa robot Karel nachádza na ľubovoľnej pozícii vo svete.

Koncová situácia

Na konci sa robot Karel nachádza na severovýchode, teda v pravom hornom rohu sveta, otočený smerom na západ.

Uloha 3

Funkcia chessboard() (4b)

Vytvorte funkciu void chessboard(), pomocou ktorej Karel dokáže vytvoriť šachovnicu ľubovoľnej veľkosti.

Čierne políčko je bez značky Biele políčko obsahuje značku. Karel bude začínať v ľavom dolnom rohu, kde je vždy políčko čierne.

Return

Funkcia nevráti žiadnu hodnotu.

Počiatočná situácia

Karel začína v ľavom dolnom rohu sveta (čierne políčko).
